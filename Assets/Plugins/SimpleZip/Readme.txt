Simple Zip is a crossplatform plugin which will help you to compress and decompress data using Zip.

Features:
- Compress data to string
- Compress data to byte array
- Decompress data
- Works on iOS, Android, WebGL and Desktop

Usage:
	Zip.Compress(PLAIN_TEXT)
	Zip.CompressToString(PLAIN_TEXT)
	Zip.Decompress(COMPRESSED_TEXT)
	Zip.Decompress(BYTE_ARRAY)
